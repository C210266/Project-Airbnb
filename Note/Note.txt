/db.json:
Để sử dụng json-server:
+ Bước 1: npm init => để tạo ra package.json
+ Bước 2: npm i json-server => để cài đặt json-server
+ Bước 3: tạo 1 file tên là db.json
+ Bước 4: Tạo lệnh start trong package.json => "start": "json-server --watch db.json --port 8000"
+ Bước 5: Dùng lệnh npm start
